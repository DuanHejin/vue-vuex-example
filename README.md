# vue-vuex-example

## 主要研究了Vuex的下面几点内容
- 概念
  - State
  - Getter
  - Mutation
  - Action
  - Module
- 实战
  - 如何集成Vuex
  - 根据state, getters, mutations, actions创建store
  - 在组件中使用mapState
  - 在组件中使用mapGetter
  - 在组件中使用mapMutation
  - 在组件中使用mapAction

### 技术栈
- Vue
- Vue-Router
- Vuex
- Bootstrap
- Sass
- Promise
